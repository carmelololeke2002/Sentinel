# Privacy Policy

## Introduction
We value your privacy and are committed to protecting your personal data. This document explains how we handle your data.

## Data Collection
- **Telemetry**: Collected only with user consent. Includes anonymized system metrics and threat statistics.
- **Logs**: Stored locally and encrypted. No sensitive data is transmitted.

## Data Usage
- Improve malware detection algorithms.
- Provide real-time threat intelligence.

## User Rights
- Opt-out of telemetry at any time.
- Request deletion of collected data.

## Security Measures
- All data transmissions are encrypted using TLS 1.3.
- Data is anonymized and aggregated before analysis.
