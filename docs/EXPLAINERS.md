# EXPLAINERS

## Architecture Decisions
1. **Rust for Agent**:
   - Chosen for its memory safety and performance.
   - Provides low-level access to system resources.

2. **React + Tauri for UI**:
   - Lightweight and modern framework.
   - Enables cross-platform compatibility.

3. **Node.js for API**:
   - Simple and scalable backend for handling updates and telemetry.

## Key Features
- **EICAR Detection**: Demonstrates the ability to detect malicious files.
- **Quarantine**: Isolates threats securely.
- **Real-Time Monitoring**: Tracks file and network activity.

## Future Enhancements
- Add machine learning for anomaly detection.
- Implement advanced sandboxing techniques.
